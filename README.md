# 📄 Resume Builder

A modern, responsive Resume Builder web application that allows users to create professional resumes, preview them in real-time, and download them as a PDF—all directly in the browser.

🌐 **Live Demo:** https://resume-builder-gamma-dusky.vercel.app/

---

## 🚀 Features

### 👤 Personal Information
- Full Name
- Email Address
- Phone Number
- Address
- LinkedIn Profile
- GitHub Profile
- Professional Summary
- Profile Photo Upload

### 🎓 Education
- Add multiple education entries
- Degree
- Institution
- Year
- CGPA/Percentage

### 💼 Experience
- Add unlimited work experiences
- Company Name
- Job Title
- Duration
- Description

### 🚀 Projects
- Add multiple projects
- Project Name
- Technology Stack
- Description
- GitHub/Live Link

### 🛠 Skills
- Add unlimited technical skills
- Remove skills instantly

### 👀 Live Resume Preview
- Updates automatically while typing
- Professional resume layout
- No page refresh required

### 📷 Profile Image Upload
- Upload JPG/PNG images
- Circular profile picture
- Instant preview

### 📄 PDF Export
- Download resume as a high-quality PDF
- Proper A4 formatting
- Print-ready layout

### 🌙 Dark Mode
- Toggle between Light and Dark themes

### 📝 Sample Resume
- Auto-fill sample data with one click

### 🧹 Clear Form
- Reset all fields instantly

### 📱 Responsive Design
- Desktop
- Tablet
- Mobile

# 📖 Usage

1. Enter your personal details.
2. Add education details.
3. Add work experience.
4. Add projects.
5. Add skills.
6. Upload your profile picture.
7. Preview updates automatically.
8. Click **Download PDF**.
9. Save your professional resume.
